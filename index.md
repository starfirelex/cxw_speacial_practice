# SQL与MySQL数据库课程

## 课程简介

本课程系统讲解SQL（结构化查询语言）和MySQL（关系型数据库管理系统）的核心概念和实际应用。通过本课程的学习，学员将掌握SQL语言的基础语法和MySQL数据库的使用方法，为后续的数据库开发和管理打下坚实基础。

## 课程特色

- **理论与实践结合**：完整讲义 + 6个实践案例
- **循序渐进**：从基础到高级，难度逐步提升
- **实战导向**：32道习题，涵盖所有知识点
- **开源免费**：基于MySQL开源数据库
- **易于上手**：适合初学者学习

## 课程内容

### 第一部分：SQL基础

1. **SQL概述**
   - SQL的定义与发展历史
   - SQL的标准化特性

2. **DDL（数据定义语言）**
   - CREATE、ALTER、DROP语句
   - 约束的使用
   - 案例1：学生信息管理系统

3. **DML（数据操作语言）**
   - INSERT、UPDATE、DELETE语句
   - WHERE条件的重要性
   - 案例2：学生数据管理

4. **DQL（数据查询语言）**
   - SELECT查询
   - WHERE条件、ORDER BY排序
   - GROUP BY分组、HAVING筛选
   - 多表连接查询
   - 案例3：学生成绩查询

5. **DCL（数据控制语言）**
   - GRANT、REVOKE权限管理
   - 用户管理
   - 案例4：用户权限管理

### 第二部分：MySQL基础

1. **MySQL概述**
   - MySQL的定义与发展历史
   - 关系型数据库特征

2. **MySQL架构特征**
   - 客户端/服务器架构
   - 开源优势

3. **MySQL性能优势**
   - 高性能：多线程、查询缓存
   - 高可用性：主从复制、集群
   - 强扩展性：存储引擎插件化
   - 案例5：存储引擎对比

4. **MySQL应用场景**
   - Web应用
   - 中小企业
   - 嵌入式应用
   - 案例6：Python连接MySQL

## 学习路径

1. **阅读课程大纲** → 了解课程整体结构
2. **学习课程讲义** → 掌握理论知识
3. **实践案例** → 动手操作加深理解
4. **完成习题** → 巩固所学内容

## 快速开始

### 环境准备

1. **安装MySQL**
   - 下载地址：https://dev.mysql.com/downloads/
   - 推荐版本：MySQL 8.0+

2. **安装Python（用于案例6）**
   ```bash
   pip install mysql-connector-python
   ```

3. **图形化管理工具（可选）**
   - MySQL Workbench
   - Navicat
   - DBeaver

- [官方文档](https://dev.mysql.com/doc/)
