# SQL与MySQL课程

## 项目简介

本项目包含SQL与MySQL数据库课程的完整教学材料，包括课程大纲、讲义、案例计划和习题集。

## 课程材料

- [课程大纲](课程内容/课程大纲.md) - 课程结构和学习路径
- [课程讲义](课程内容/SQL与MySQL课程讲义.md) - 完整理论讲解
- [案例计划](课程内容/案例计划.md) - 6个实践案例
- [习题集](课程内容/习题集.md) - 32道练习题

## 在线访问

课程网站：[https://github.com/Kailei111/sql-mysql-course]

## 课程结构

### 第一部分：SQL基础
- SQL概述
- DDL（数据定义语言）
- DML（数据操作语言）
- DQL（数据查询语言）
- DCL（数据控制语言）

### 第二部分：MySQL基础
- MySQL概述
- MySQL架构特征
- MySQL性能优势
- MySQL应用场景

## 学习路径

1. 阅读课程大纲，了解整体结构
2. 学习课程讲义，掌握理论知识
3. 实践案例，动手操作
4. 完成习题，巩固知识

## 环境要求

- MySQL 8.0+
- Python 3.7+（用于案例6）
- 图形化管理工具（可选）
